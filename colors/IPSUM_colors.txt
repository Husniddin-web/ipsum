   CMYK: 0 0 0 40
   RGB: 169 171 174
   HEX: #a9abae
   Pantone: 429 C

Implementation notes:
- Use #ed3237 and #35374b as the main brand pair.
- Use #f7adaf, #8c9aab, and #a9abae as supporting colors.
- For web UI, prefer HEX/RGB values.
- For print or design handoff, use CMYK and Pantone values.
